# Primeros pasos

## Estado actual

El proyecto está en fase de base arquitectónica. La estructura ya separa motor Excel, backend, frontend y esquemas compartidos.

## Qué puedes hacer ahora

- revisar la documentación
- explorar la estructura del repositorio
- extender el motor Excel
- empezar el backend o la interfaz web

## Qué no incluye todavía

- autenticación real
- frontend funcional
- base de datos conectada
- exportación completa

## Siguiente hito recomendado

Implementar el modelo de dominio del backend:

- usuarios
- ejercicios
- variantes
- sesiones
- series
- progresión
