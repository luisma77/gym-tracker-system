# Gym Tracker Web App

Aplicacion web del proyecto `Gym Tracker`, pensada para registrar sesiones completas de entrenamiento siguiendo la logica del Excel original.

## Que incluye

- autenticacion de usuario
- registro de sesiones PPL por dia completo
- sugerencias de progresion por ejercicio segun reps, RIR y ultima sesion
- control de volumen semanal por grupo muscular
- registro de medidas corporales
